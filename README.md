# Python Console Mini Projects

This repository contains several console applications developed while learning Python fundamentals.

## Projects

### Random Number Game
A simple guessing game with hints and sound feedback.

### Library Management System
Allows adding, searching and displaying books using Python dictionaries.

### Inventory Management System
Supports product management, stock reports and saving data to text files.

## Features

- Colored console output
- Text-to-speech messages
- Sound effects
- File persistence
- Inventory statistics

## Technologies

- Python
- pygame
- colorama
- pywin32

## Run

```bash
pip install -r requirements.txt
python main.py
